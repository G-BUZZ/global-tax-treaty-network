# Key Insights

This document summarizes the main findings of the current global tax treaty network analysis.

## Main findings

- France and the United Kingdom emerge as the most connected jurisdictions by degree.
- Switzerland appears highly central, including as an intermediary node.
- Italy is strongly connected, but it does not dominate the network once the analysis is scaled beyond the Italy-centered seed graph.
- The global network appears to be organized around multiple fiscal hubs rather than a single dominant jurisdiction.

## Interpretation note

The findings describe network structure only. They should not be interpreted as legal, tax, financial, or investment advice.
