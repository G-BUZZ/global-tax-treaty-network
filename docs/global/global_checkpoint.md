# Global Tax Treaty Network — Checkpoint

## Current dataset
- Source: OECD treaty-based withholding tax rates export
- Network type: undirected treaty relationship network
- Nodes: 192
- Edges: 2793

## Key observations
- France and the United Kingdom emerge as the most connected jurisdictions by degree.
- Switzerland also appears highly central, especially as an intermediary node.
- Italy is strongly connected, but no longer dominates the network when the analysis is expanded globally.
- The network reveals multiple European and international fiscal hubs rather than a single-country-centered structure.

## Why this matters
This project moved from a manually curated Italy-centered seed network to a global treaty network derived from OECD data.
It now supports comparative network analysis across a large set of jurisdictions.
