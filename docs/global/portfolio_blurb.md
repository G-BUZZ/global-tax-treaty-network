# Portfolio Blurb

Global Tax Treaty Network is a network analysis project that models bilateral tax treaty relationships between jurisdictions using OECD-derived data.

I built a global graph of 192 jurisdictions and 2793 undirected treaty relationships, produced node and edge datasets, calculated centrality metrics, and generated visual outputs to identify highly connected jurisdictions and structural patterns across the network.

The project combines data cleaning, relational modeling, network analysis, and visual communication.
