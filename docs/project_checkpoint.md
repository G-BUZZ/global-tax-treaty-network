# Project Checkpoint

## Current stage
This is an early methodological checkpoint of the Global Tax Treaty Network project.

## What has been completed
- project structure created
- nodes and edges datasets initialized
- first Italy-centered treaty network built
- data validation script completed
- network summary script completed
- centrality export script completed
- first technical and showcase visual outputs generated

## Current limitation
The current graph is still an ego-network centered on Italy.
This means Italy dominates all centrality metrics by construction.

## Why this still matters
This stage proves that the workflow works:
CSV -> validation -> network construction -> centrality analysis -> PNG export

## Next milestone
Expand the Italy treaty network with many more partner jurisdictions and then move toward a broader multi-country network.
