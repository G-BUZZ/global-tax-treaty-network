# Raw source exports

Raw source exports are intentionally kept local and are not committed to the public repository.

This avoids redistributing full third-party source exports while preserving reproducibility through:

- documented source attribution;
- processing scripts;
- derived outputs committed in the repository;
- checksum manifest for local verification.

If a raw file is needed to reproduce the analysis, place it under `data/raw/` using the same filename shown in the checksum manifest.
